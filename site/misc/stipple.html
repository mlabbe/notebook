# Mike's Viz Notebook # Back to [table of contents](../index.html) # Encoding Stipple Patterns in Bits # As famously used by [glLineStipple](https://registry.khronos.org/OpenGL-Refpages/gl2.1/xhtml/glLineStipple.xml), a stipple pattern can be decoded from a sequence of bits. Each subsequent bit determines whether the next $n$ pixels are on or off. This corresponds to the `factor` parameter in `glLineStipple()`. # Interactive Demo #
# Pseudocode # ~~~~~~~~~~~~~ c u16 pattern = 0x5555; u16 mask = 0x8000; // set leftmost bit for (int i = 0; i < 16; i++) { int bit = (pattern & mask) != 0; if (bit) set_pixel(); mask >>= 1; } ~~~~~~~~~~~~~ [back](../index.html)
