# Mike's Viz Notebook # Back to [table of contents](../index.html) # Split Vector into Parallel and Perpendicular Parts # It's possible to "sift" the parallel and perpendicular components of a vector, using projection against a constant unit vector. Once you have the parallel component with $b_{||} = (\hat{a} \cdot \vec{b})\hat{a}$ you can get the perpendicular component with $b_{\perp} = \vec{b} - \vec{b}_{||}$, $b_{\perp} = \vec{b} - (\hat{a} \cdot \vec{b})\hat{a}$ # Interactive Demo #
# Pseudocode # ~~~~~~~~~~~~~ c // derive b's parallel vector vec2 a_norm, b, out_b_parallel; { // assume a is normalized real a_dot_b = dot(a_norm, b); out_b_parallel = mult(a, a_dot_b); } // derive b's perpendicular vector vec2 b, b_parallel, out_b_perpendicular; { out_b_perpendicular = sub(b, b_parallel); } ~~~~~~~~~~~~~ [back](../index.html)
