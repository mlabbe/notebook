# Magnitude # If you dot a vector by itself you get the length of the projection $||v||$ times the length of the vector you are projecting onto, which is also $||v||$. $\vec{v} \cdot \vec{v} = ||v||^2$ or $||v|| = \sqrt{v \cdot v}$ # Interactive Demo #
# Pseudocode # ~~~~~~~~~~~~~ c vec2 a; real out_magnitude; { out_magnitude = sqrt(dot(a, a)); } ~~~~~~~~~~~~~ [back](../index.html)
