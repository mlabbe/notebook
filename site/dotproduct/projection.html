# Projection Property of a Dot Product # For a given vector $\vec{b}$, project it onto unit vector $\hat{a}$. $\hat{a} \cdot \vec{b}$ produces the *signed* length that can be scaled by $\hat{a}$ to produce a vector that is the length of $\vec{b}$, but with the orientation of $\hat{a}$. $p = \hat{a} * (\hat{a} \cdot \vec{b})$ Note that dot product is commutative; the order of the vectors in the product do not matter: $\vec{a} \cdot \vec{b} = \vec{b} \cdot \vec{a}$ # Interactive Demo #
# Pseudocode # ~~~~~~~~~~~~~ c // project a onto b vec2 a, b, out_projected; { // assumes magnitude of a is > 0 a = normalize(a); real signed_length = dot(a, b); out_projected = mul(a, signed_length); } ~~~~~~~~~~~~~ [back](../index.html)
