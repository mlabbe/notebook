# Halfspaces # The sign of $\hat{a} \cdot b$ tell us which halfspace $b$ lies in. If $\hat{a} \cdot b > 0$, $b$ is front-facing. Otherwise, it is back-facing.
## C Pseudocode ## ~~~~~~~~~~~~~ c // determine which halfspace of a that b is in vec2 a, b; bool out_is_front_facing; { a = normalize(a); real a_dot_b = dot(a, b); out_is_front_facing = a_dot_b >= 0; } ~~~~~~~~~~~~~ [back](../index.html)
